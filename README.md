# Swami Gift Card Shortcode

A lightweight WordPress plugin that provides a customizable gift card form using a simple shortcode.

---

## 🚀 Description

This plugin allows you to display a beautiful gift card form on any WordPress page or post using a shortcode.  
It is designed for Elementor and standard WordPress editor compatibility.

---

## ✨ Features

- 🎁 Gift card form shortcode
- ⚡ Lightweight and fast
- 🧩 Elementor compatible
- 🎨 Custom styling support (assets folder)
- 🔌 Easy integration with any theme
